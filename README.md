# LLM Oncologic Summarization Project

## Overview
The LLM Oncologic Summarization Project aims to leverage large language models (LLMs) to generate concise and accurate summaries of oncological literature. This project is designed to enhance the accessibility of crucial information for healthcare professionals and researchers.

## Project Structure
- `src/`: Contains the source code for model training and evaluation.
- `data/`: Holds datasets used for training and testing the model.
- `notebooks/`: Jupyter notebooks for exploratory data analysis and results visualization.
- `README.md`: This documentation file.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Robertjam954/llm_summarization_br_ca.git
   cd llm_summarization_br_ca
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Ensure you have access to the necessary datasets that can be placed in the `data/` directory.
4. Run the training scripts located in the `src/` directory to start the model training process.

## Data Privacy Notes
All data utilized in this project is handled with the utmost respect for privacy and confidentiality. The datasets used are sourced from publicly available resources or are anonymized to prevent the identification of any individual subjects. Users are encouraged to adhere to ethical guidelines and regulations regarding data privacy when utilizing any information generated by this project.
